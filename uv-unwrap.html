Blender UV Unwrap (전개도 펼치기) 완벽 가이드

오늘의 목표: "왜곡 없는 완벽한 에셋을 유니티로!"

멋지게 만든 3D 모델을 유니티로 가져갔을 때 텍스처가 찌그러지거나 그림자가 깨진 적이 있나요?
그 이유는 바로 UV 언랩(Unwrap) 과정이 생략되었거나 잘못되었기 때문입니다. 오늘 강의에서는 3D 모델링의 필수 관문인 UV의 기초 개념부터, 유니티 실시간 렌더링 엔진으로 넘기기 전 반드시 거쳐야 하는 최적화 작업까지 실습을 통해 확실히 익혀보겠습니다.

UV 매핑의 정확한 개념

3D 공간의 입체적인 모델에 그림(텍스처)을 입히려면, 모델의 표면을 평평한 2D 평면으로 쫙 펴주어야 합니다. 이 전개도를 만드는 작업을 UV 언랩(Unwrap)이라고 부릅니다. 3D 공간을 X, Y, Z로 부르듯이, 2D 텍스처 공간의 가로 세로를 U와 V3D 축인 X,Y,Z와 헷갈리지 않도록 텍스처 전용 좌표계로 U(가로), V(세로)를 사용합니다.로 부릅니다.

2D 전개도 (U,V 좌표)

1. 왜곡 없는 텍스처링

UV가 제대로 펴져 있지 않으면 페인팅을 할 때 이미지가 심하게 찌그러지거나 늘어납니다.

2. 라이트맵 베이킹 (필수)

유니티에서 빛 연산을 텍스처로 구워낼 때(Baking), UV 면이 조금이라도 겹치면 그림자가 깨져 검은 얼룩이 발생합니다.

3. 게임 엔진 최적화

게임 엔진은 한정된 텍스처 메모리를 사용합니다. 전개도를 낭비 없이 꽉 채워(Packing) 성능을 끌어올려야 합니다.

블렌더에서 UV 작업을 시작하려면 반드시 지켜야 하는 기본 3단계가 있습니다. 이 순서를 지키지 않으면 좌측 UV 창에 아무것도 나타나지 않아 당황하게 됩니다.

Color Grid의 절대적인 중요성

새로운 텍스처를 생성할 때 'Color Grid' (또는 UV Grid) 옵션을 꼭 켜주세요. 모델에 이 체스판 무늬를 입혀보면 눈으로 즉시 UV의 상태를 진단할 수 있습니다.

  • 정상: 체크무늬가 모델 전체에 일정한 크기의 '정사각형'으로 보임
  • 오류: 무늬가 직사각형으로 길게 늘어지거나 물결처럼 찌그러짐 (왜곡 발생)
테스트 패턴

Mark Seam (재봉선 표시)

기본 큐브를 넘어 실린더나 사람 옷처럼 복잡한 모델을 펼치려면, 블렌더에게 "어디를 가위로 자를지" 알려주어야 합니다. 자르고자 하는 모서리(Edge)를 선택하고 우클릭 > Mark Seam을 누르면 붉은색 절취선이 생성됩니다.

단축키: U 누른 후 메뉴에서 Mark Seam 클릭
Seam
실린더 (원기둥) 언랩

음료수 캔을 종이로 펼친다고 상상해 보세요. 뚜껑과 몸통을 분리해야 합니다.

  1. 1
    위아래 뚜껑 분리: 원형 윗면과 아랫면의 테두리 엣지(루프)를 모두 선택하여 Seam을 지정해 뚜껑을 떼어냅니다.
  2. 2
    몸통 가르기: 원통형 몸통이 평평하게 펴지도록, 눈에 잘 띄지 않는 뒷면의 세로 엣지 하나를 선택해 Seam을 그어줍니다.
  3. 3
    전체 선택 후 언랩(U)을 하면 완벽한 직사각형 몸통과 2개의 원형 뚜껑 전개도가 완성됩니다.
구체 (Sphere) 언랩

완벽한 구형을 왜곡 없이 2D 평면으로 펴는 것은 불가능합니다. (지구본을 세계지도로 만드는 것과 동일)

  • 자동: Sphere Projection

    블렌더가 자동으로 펼쳐주지만, 양쪽 끝 극점(Pole) 부분에 심한 왜곡이 생깁니다. 자연물에 적합합니다.

  • 수동 Seam 방식

    게임 에셋에서는 주로 카메라에 안 보이는 뒷면 절반에 Seam을 넣고 억지로 펼친 후, 끊어진 부분을 텍스처 페인트로 자연스럽게 덮어 숨깁니다.

UV 맵핑 시뮬레이터
관찰 포인트 👀

원통의 몸통은 완벽한 사각형으로 펴져 왜곡이 없지만, 위아래 원형 뚜껑과 만나는 곳은 어쩔 수 없이 자르는 선(Seam)이 필요함을 우측 전개도에서 확인할 수 있습니다.

블렌더의 기본 구체(Sphere) 언랩입니다. 3D 모델의 극점(Pole) 부분과 우측 2D 전개도를 비교해보면, 이미지가 극심하게 왜곡되어 모이는 현상을 관찰할 수 있습니다.

큐브를 둥글게 깎아 만든 구체입니다. 6개의 면으로 분할하여 펼쳤기 때문에 우측 전개도를 보면 6개의 조각(Island)으로 분리된 것을 확인할 수 있습니다.

UV 패킹 (Packing)

잘라낸 여러 개의 UV 조각들을 0에서 1 사이의 한정된 정사각형(UV Space) 안에 빈틈없이 퍼즐 맞추듯 꽉 채워 배치해야 텍스처 해상도 손실을 막을 수 있습니다.

마진 (Margin) 유지

UV 조각들을 너무 바짝 붙이면, 유니티에서 카메라가 멀어질 때(Mipmap 적용 시) 옆 조각의 색깔이 번져 들어오는 현상(Bleeding)이 생깁니다. 반드시 약간의 여백(Margin)을 두세요.

Seam은 보이지 않는 곳에

가위질을 한 재봉선(Seam) 부분은 아무리 텍스처를 잘 칠해도 묘하게 끊어져 보일 위험이 큽니다. 모델의 바닥면, 뒷면, 안쪽 등 플레이어의 시선이 잘 닿지 않는 곳으로 Seam을 숨기세요.

완벽한 UV 위에 그리는 'Texture Paint'

지금까지 진행한 깔끔한 UV 언랩은 수준 높은 텍스처링의 절대적인 전제 조건입니다. UV가 겹쳐있거나 왜곡되어 있다면 한 곳에 브러시를 칠해도 물감이 엉뚱한 곳에 묻어나게 됩니다.

이제 블렌더의 Texture Paint 모드로 넘어가 모델 위에 직접 색상을 칠하고, 완성된 이미지 파일(.png)과 모델 데이터(.fbx)를 유니티로 넘기면 렌더링 준비가 끝납니다!
