기말 프로젝트 공간 빌딩 워크샵

순서에 따라 팀원들과 함께 프로젝트 기획을 구체화해 보세요.

피그잼(FigJam)에 모여 우리 조가 만들 세계관의 큰 그림을 맞추는 단계입니다.

1. 내 키워드 꺼내기

사전 조사에서 본인이 제출했던 '공간 컨셉'과 '희망 역할(기획/3D/로직/영상 등)'을 조원들에게 공유합니다. 서로의 교집합을 찾아보세요.

2. 공간 설정 체크리스트

우리 조의 방향성과 가장 잘 맞는 항목을 카테고리별로 1~2개씩 선택해 합의해 보세요.

시간대 & 날씨

  • ☀️ 청명한 낮 (눈부신 자연광, 선명한 그림자)
  • ⛅ 흐린 날 (빛바랜 색감, 차분하고 부드러운 빛)
  • 🌅 해 질 녘 (황금빛 노을, 길게 늘어지는 그림자)
  • 🌆 매직 아워 (보랏빛/오렌지빛이 섞인 하늘)
  • 🌙 깊은 밤 (은은한 달빛, 짙은 어둠의 대비)
  • 🌃 화려한 밤 (네온사인, 다채로운 인공 조명)
  • 🌫️ 새벽 안개 (시야가 제한된 몽환적인 분위기)
  • 🌧️ 비/폭풍우 (우중충함, 물에 젖은 바닥 반사)
  • 🌌 우주/초현실 (물리법칙을 벗어난 비현실적 광원)

시각적 테마

  • 🌿 힐링 정원 / 숲 (식물, 동양풍, 몽글몽글한 자연)
  • 🌊 심해 / 물의 공간 (수중 도시, 고요한 바다)
  • ⚙️ 사이버펑크 (화려한 네온, 복잡한 기계 도시)
  • 🕰️ 스팀펑크 / 인더스트리얼 (톱니바퀴, 낡은 금속)
  • 📺 Y2K / 레트로 (낡은 전자기기, 90년대 캠코더 감성)
  • 🍭 키치 / 테마파크 (알록달록, 장난감, 놀이공원)
  • 🏰 환상 / 동화 (마법, 거대한 사물, 앨리스의 세계)
  • 🏚️ 포스트 아포칼립스 (자연이 덮은 폐허, 생존)
  • 🔮 리미널 스페이스 (무한 반복, 묘하게 기괴한 공간)
  • 🏛️ 모던 / 미니멀 갤러리 (깔끔한 직선, 여백의 미)
  • ☁️ 천상 / 신화적 (구름 위 성전, 성스럽고 초월적인)
  • 🩸 다크 / 고딕 미스터리 (으스스한 저택, 그림자)

방문자 핵심 감정

  • ☕ 편안함과 위로 (마음이 차분해지는 힐링)
  • 😲 경이로움과 압도감 (스케일에 감탄하는 경험)
  • 🎢 호기심과 모험심 (계속 탐험하고 싶은 기대감)
  • 🎉 신남과 해방감 (현실의 스트레스를 벗어나는 즐거움)
  • 🎞️ 아련함과 향수 (과거를 회상하게 되는 뭉클함)
  • 😔 고독감과 쓸쓸함 (세상에 혼자 남겨진 듯한 여운)
  • 🥶 긴장감과 오싹함 (미지의 공간이 주는 스릴)
  • 🤔 신비로움 (현실인지 꿈인지 헷갈리는 몽환)

3. 레퍼런스 수집 및 1줄 로그라인

📌 각자 합의된 분위기에 맞는 레퍼런스 이미지(핀터레스트 등)를 1~2장씩 찾아 피그잼에 수집합니다. (AI 이미지 믹싱 툴을 활용해도 좋습니다.)

📌 수집된 이미지와 키워드를 조합하여 우리 조를 소개할 '1줄 로그라인'을 완성합니다.

" (        )에서 (        )하는 (        ) 공간 "

교수님 안내에 따라 진행됩니다!

모든 조의 '테마 예고편(로그라인과 이미지)'이 메인 스크린에 전시됩니다. 전체 테마를 확인한 후, 본인의 강점과 가장 잘 맞는 조로 자율적인 이동을 시작합니다.

  • 이동 시 조 인원은 최소 3명 ~ 최대 5명으로 제한합니다.
  • 역할(기획/3D/로직 등)이 너무 겹치지 않는지 꼭 확인하세요.

1. 새로운 팀원 맞이하기 (3문장 자기소개)

팀 이동이 끝난 후 확정된 팀원들과 가볍게 인사를 나눕니다.

1️⃣ "내 이름은 [ OOO ] 입니다."
2️⃣ "우리 조 테마에서 가장 마음에 들었던 부분(또는 합류한 이유)은 [ OOO ] 입니다."
3️⃣ "나는 이번 프로젝트에서 [ 기획 / 3D / 로직 / 영상 등 ] 역할을 맡아 기여하고 싶습니다."

2. 디테일 요소 무작위 발산 (포스트잇)

공간을 채울 구체적인 요소들을 피그잼 포스트잇에 자유롭게 적어냅니다. (질보다 양!)

건축물/자연물
소품(오브젝트)
사운드/BGM
인터랙션(상호작용)

1. 어피니티 다이어그램 (아이디어 묶기)

앞 단계에서 흩뿌려진 포스트잇 중 버릴 것은 버리고, 비슷한 요소끼리 드래그하여 그룹으로 묶어 공간의 구성 요소를 확정합니다.

2. 탑다운(Top-down) 2D 맵 스케치

정리된 요소를 바탕으로 유니티(Unity) 씬에 올릴 맵의 평면도를 피그잼이나 종이에 그립니다.

✅ 스케치 필수 포함 요소

  • Spawn Point (시작점): 유저가 처음 공간에 들어왔을 때의 위치
  • 유저 동선 (Path): 유저가 둘러보길 바라는 주 이동 경로 (화살표)
  • 메인 랜드마크: 공간의 주제를 보여주는 가장 큰 오브젝트의 위치
  • 바운더리(가벽): 맵의 끝을 막아줄 장치 (절벽, 울타리 등)

스케치한 맵을 구현하기 위해 당장 만들어야 할 준비물 목록을 작성하고 역할을 나눕니다.

🎨 3D 에셋 (모델링) 예: 메인 오두막 외관 모델링 예: 김메타
예: 로우폴리 나무 3종 세트 예: 박블렌

예: 전체 조명(Skybox) 및 지형 세팅 예: 최유닛
📝 기타 (영상/디자인 등) 예: BGM 서치 및 오프닝 UI 기획 예: 정기획

교수님 안내 사항

워크샵이 종료되면 오늘 작업한 피그잼 보드 결과물(팀명, 맵 스케치, 역할 분배표 등)을 캡처하여 정해진 곳(LMS 등)에 업로드 후 퇴실해 주시기 바랍니다. 수고하셨습니다!
