Blender UV Unwrap (전개도 펼치기) 마스터하기

학습 목표: UV 매핑이란?

블렌더에서 만든 3D 모델을 Unity나 Unreal Engine으로 가져가면 처음에는 색이 없는 찰흙 덩어리로 보입니다. 여기에 텍스처(이미지)를 입히려면 입체적인 3D 표면을 가위로 잘라 평평한 2D 평면으로 펼쳐야 합니다. (택배 상자를 펼치는 것과 같습니다.)

공간의 좌표를 X, Y, Z로 부르듯, 텍스처 전용 2D 좌표를 U와 V라고 부르기 때문에 이 과정을 'UV 매핑' 또는 'UV 언랩(Unwrap)'이라고 합니다.

실습 준비: 기본 도형 배치
  1. 블렌더를 켜면 중앙에 기본 큐브(Cube)가 있습니다.
  2. Shift + A를 눌러 실린더(Cylinder)와 UV 구체(UV Sphere)를 각각 추가합니다.
  3. 도형들이 겹치지 않게 G (이동) ➔ X (X축 고정)를 눌러 나란히 배치해 둡니다.

작업 화면 분할하기
  • 블렌더 최상단 메뉴 탭에서 [UV Editing]을 클릭합니다. 화면이 좌측(2D 전개도)과 우측(3D 뷰포트)으로 나뉩니다.
  • 우측 3D 뷰포트에서 기본 큐브를 선택하고 Tab 키를 눌러 에디트 모드(Edit Mode)로 진입합니다.
  • A 키를 눌러 큐브의 전체 면을 선택해 봅니다. 좌측 2D 화면에 십자가 모양의 전개도가 나타납니다.
왜곡 확인을 위한 Color Grid 생성 및 적용

펼쳐진 전개도가 찌그러지지 않았는지 눈으로 쉽게 확인하기 위해 체스판 형태의 테스트 이미지를 입혀보겠습니다. 정사각형 무늬가 직사각형으로 늘어난다면 UV가 왜곡(Distortion)된 것입니다.

  1. 좌측 UV 에디터 상단 메뉴에서 New Image를 클릭합니다. (단축키 Alt + N)
  2. 이름을 'test_grid'로 설정하고, Generated Type을 'UV Grid'로 변경한 후 OK를 누릅니다.
  3. 우측 속성 패널 하단의 빨간색 지구본 아이콘인 Material Properties 탭으로 이동합니다.
  4. Base Color 옆의 🟡 노란색 점을 클릭하고 Image Texture를 선택합니다.
  5. 좌측의 그림 아이콘을 눌러 방금 만든 'test_grid'를 연결해 줍니다.
  6. 우측 3D 뷰포트 상단 우측의 동그란 아이콘 중 3번째인 Material Preview 모드를 켜서 큐브 표면에 격자가 정상적으로 나오는지 확인합니다.

3D 뷰포트에서 전체 선택(A) 후 U 키를 누르면 다양한 UV 매핑 방식이 나옵니다.

  • ✅ Unwrap (기본): 우리가 설정한 가위질 선(Seam)을 기준으로 면을 펼칩니다. 가장 많이 쓰입니다.
  • ✅ Smart UV Project: 블렌더가 각도를 계산해 자동으로 잘라 펼칩니다. 복잡한 기계나 배경 프랍에 유용하지만 전개도가 지저분해질 수 있습니다.
  • ⚠️ Cube Projection 등: 형태에 맞춰 강제로 투영시킵니다. Unity 라이트맵 베이킹 시 면이 겹쳐 오류가 날 수 있어 주의가 필요합니다.
실시간 UV 왜곡 시뮬레이터
관찰 포인트 👀

원통의 몸통은 완벽한 사각형으로 펴져 왜곡이 없지만, 위아래 원형 뚜껑과 만나는 곳은 어쩔 수 없이 자르는 선(Seam)이 필요함을 우측 전개도에서 확인할 수 있습니다.

블렌더의 기본 구체(Sphere) 언랩입니다. 3D 모델의 극점(Pole) 부분과 우측 2D 전개도를 비교해보면, 이미지가 가로로 극심하게 왜곡되어 모이는 현상을 관찰할 수 있습니다.

큐브를 둥글게 깎아 만든 구체입니다. 6개의 면으로 분할하여 펼쳤기 때문에 우측 전개도를 보면 6개의 독립된 조각으로 분리된 것을 확인할 수 있습니다.

텍셀 밀도(Texel Density) 불균형: 동일한 3D 면적임에도 윗면(Top)은 UV 영역이 너무 커 패턴이 비정상적으로 촘촘하게 보입니다. 반면 우측면(Right)은 UV 영역이 너무 작아 텍스처 해상도가 심하게 깨집니다. 마우스로 회전하며 각 면을 비교해 보세요!

드래그로 3D 회전
2D 전개도 (UV 맵)
재봉선(Seam)을 긋는 3대 원칙
  • 1. 시선이 닿지 않는 곳에 숨기기 전개도가 끊어지면 텍스처도 끊어질 확률이 높습니다. 사물의 바닥면, 뒷면 등 플레이어에게 잘 안 보이는 곳을 우선적으로 자릅니다.
  • 2. 90도로 심하게 꺾이는 모서리 각진 하드 서페이스(기계 등)에서 모서리를 자르지 않고 억지로 펴면, 유니티 라이트맵을 구울 때 그림자가 까맣게 타버립니다.
  • 3. 실제 사물의 이음매 실제 옷의 재봉선, 가죽 소파의 꿰맨 자국 등 원래 물리적으로 끊어져 있는 부위를 Seam으로 만들면 자연스럽습니다.

UV 아일랜드 (Island)란?

가위질(Seam)을 통해 독립적으로 잘려져 나온 2D 전개도 조각들 하나하나를 지칭합니다.

면적이 불균형할 때: 텍셀 밀도 맞추기

3D 공간에서는 면적이 비슷한데, UV 전개도에서 조각(아일랜드)의 크기가 서로 다르면 체크무늬 크기(해상도)가 부위마다 달라져 이질감이 발생합니다.

1초 해결 팁

UV 에디터 창에서 전체 선택(A) 후, 상단 메뉴에서 UV > Average Islands Scale을 클릭하세요. 3D 모델의 실제 표면적 비율에 맞춰 UV 조각들의 크기가 자동으로 균일해집니다!

기본 원리를 배웠으니, 실전 프랍인 '책(Book)'을 만들어 직접 언랩해 보겠습니다.

책 뼈대 모델링
  1. 새로운 큐브를 추가하고 S (스케일) 키를 눌러 책처럼 납작하고 넓적하게 만듭니다.
  2. Tab 으로 에디트 모드에 들어갑니다.
  3. 종이(속지)가 될 세 면(위, 아래, 앞)을 선택합니다.
  4. I (인셋) 키를 눌러 면을 약간 안으로 밀어 넣습니다.
  5. E (돌출) 키를 눌러 안으로 쑥 파이게 만들어 겉표지와 종이를 구분해 줍니다.

초보자 최다 실수: Scale Apply

Object Mode에서 스케일(S)을 조절하여 책을 만들었다면, 블렌더는 여러분의 책을 "여전히 정사각형 큐브"로 착각합니다. 이 상태로 언랩을 하면 UV가 원본 큐브 비율에 맞춰져 심하게 찌그러집니다.

해결: Object Mode에서 모델 선택 ➔ Ctrl + A 누르기 ➔ Scale 클릭!
책의 이음새(Seam) 마킹과 Unwrap

겉표지와 종이 부분이 분리되도록 가위질 선을 지정합니다.

  1. 에디트 모드에서 숫자 2를 눌러 선(Edge) 선택 모드로 바꿉니다.
  2. 겉표지와 속지가 만나는 경계선들을 Shift를 누른 채 둘러가며 모두 선택합니다.
  3. 우클릭 후 'Mark Seam (씨임을 마크)'을 클릭합니다. (선이 빨갛게 변합니다)
  1. A를 눌러 모델 전체를 다시 선택합니다.
  2. U를 누르고 'Unwrap'을 클릭합니다.
  3. 결과 확인: 좌측 UV 에디터를 보면 표지와 속지가 깔끔하게 분리되어(여러 개의 아일랜드로) 펼쳐진 것을 볼 수 있습니다.
텍스처 아틀라스(Texture Atlas)란?

해리포터 마법 도서관처럼 수천 권의 책을 Unity에 배치한다고 가정해 봅시다. 각기 다른 색을 내기 위해 매테리얼을 수천 개 만들면, 화면을 그릴 때마다 엔진 과부하로 '드로우 콜(Draw Call)'이 치솟아 게임 프레임이 심하게 끊깁니다.

이를 해결하는 실무 마법이 바로 '텍스처 아틀라스'입니다. 여러 색상과 질감을 큰 이미지 1장에 모아두고(컬러 팔레트), 수많은 오브젝트가 단 1개의 매테리얼만 공유하게 만드는 최적화 기법입니다.

실전: 아틀라스 매핑 클릭-바이-클릭 가이드

1. 세팅 및 다중 편집 진입
  1. 오브젝트 복제: Object 모드에서 완성된 책을 선택하고 Shift + D를 눌러 3권으로 복제해 나란히 둡니다.
  2. 단일 매테리얼 세팅: 3권의 책에 적용된 기존 매테리얼을 모두 지우고, 'Atlas_Mat'이라는 매테리얼을 딱 1개만 생성하여 모두 동일하게 적용합니다.
  3. 'Atlas_Mat'의 베이스 컬러에 미리 준비한 다양한 컬러 칸이 있는 팔레트 이미지를 연결합니다.
  4. 다중 에디트 진입: 3권의 책을 한 번에 다중 선택(Shift 클릭)하고 Tab 키로 에디트 모드에 들어갑니다.
2. UV Sync 및 컬러 할당 중요!
  1. 좌측 UV 에디터 상단의 🔁 'UV 동기화(UV Sync Selection)' 버튼을 켭니다. (이를 켜야 우측에서 선택한 면만 좌측에서 볼 수 있습니다.)
  2. 우측 3D 뷰포트에서 첫 번째 책의 '겉표지' 면들만 선택합니다.
  3. 좌측 UV 창에 표지 전개도(아일랜드)가 나타나면, A로 선택하고 S를 눌러 점처럼 아주 작게 축소합니다.
  4. G를 눌러 팔레트 이미지의 '빨간색' 칸에 쏙 넣습니다.
  5. 두 번째, 세 번째 책 표지도 위와 같이 각각 '파란색', '초록색' 칸에 올려줍니다. 종이 부분은 '흰색'에 올려줍니다.

🎉 축하합니다! 완벽하게 렌더링 준비가 끝났습니다.

겉보기에는 빨간 책, 파란 책 등 화려해 보이지만, 엔진 내부적으로는 '단 1장의 이미지'만 읽고 있는 아주 가볍고 최적화된 상태가 되었습니다. 유니티에서는 이 책들을 한 번에 렌더링하는 '배칭(Batching)'을 통해 프레임 드랍을 막을 수 있습니다.

💡 3줄 요약 체크리스트
  • 형태 변형 후엔 반드시 스케일 적용 (Ctrl+A)
  • 안 보이는 곳에 영리하게 Seam (재봉선) 자르기
  • 아틀라스/팔레트를 활용해 매테리얼 개수 최소화
